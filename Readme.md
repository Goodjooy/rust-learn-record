# Rust 的学习笔记

## 简介

Rust 是一门赋予每个人构建可靠且高效的软件的语言

> ps: 虽然 rust 是一门优秀的编程语言，但是我仍旧不推荐入门编程使用 rust
> 只有被其他语言的 段错误、空指针异常、迷惑性的默认行为洗礼过后，才会
> 体会到 rust 的温柔与强大 <del>和被编译器变成不会排查异常的废物</del>
